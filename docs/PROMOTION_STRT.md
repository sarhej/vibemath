## Elevator pitch (2–3 sentences)

**Vibemath** is a free, open-source web calculator that compares the monthly cost of AI-assisted software delivery across coding subscriptions (Cursor, Claude, Copilot, Windsurf), direct API usage, self-hosted hardware (Macs, RTX rigs), and rented cloud GPUs. You set your target output (lines of code, work hours, token usage), and it shows break-even points and total cost so you can choose the stack that fits your budget and throughput.

## What the tool compares

### Subscriptions (with tiers and quotas)

- **Cursor:** Pro, Pro+, Ultra (unit-based quotas, overage)
- **Claude:** Pro, Max 5x, Max 20x (soft fair-use; dashed in UI)
- **Copilot:** Pro, Biz (unit-based)
- **Windsurf:** Pro (unit-based)
- Option to hide “dotted” (soft-quota) plans for clearer comparison

### API providers

- Claude (Anthropic), GPT (OpenAI), Gemini (Google), DeepSeek, OpenRouter (blended)
- Context-heavy token pricing (input-heavy usage)

### Self-hosted (home hardware)

- **GPUs:** RTX 3060 12GB → RTX 5090, Dual RTX 4090, Custom AI Cluster
- **Macs:** Mac mini 64GB, MacBook Pro Max, Mac Studio (128GB), Mac Studio Ultra (192GB)
- Cost = monthly amortization (configurable years) + electricity × usage hours
- Productivity multiplier per model (weaker local models = lower effective throughput)

### Rented GPUs

- **Hyperscalers:** AWS H100, Azure A100 80GB, GCP L4
- **Marketplaces:** RunPod (A100, L40S), Vast.ai A100, Lambda H100

### Coding models (vibe-coder ranked)

- From Claude Opus 4.6, GPT-5.4, Gemini 3.1 Pro down to StarCoder2
- Checkboxes to include only “approved” models; VRAM and productivity % shown per option

## Features (from the app)

- Full-width cost-vs-output chart (LOC, tokens, hours/month on X; monthly cost USD on Y)
- Target LOC/month, work hours/month, lines per hour, parallel AI workstreams (with diminishing returns)
- Tokens per net LOC (incl. context) with presets (e.g. “Heavy Cursor”, “Vibe God”)
- Productivity templates and “sync to throughput”
- Shareable URL (compact encoded state) and **Save locally** profiles (name, load, share, delete)
- PNG export of the chart
- Home-hosted assumptions: amortization years, electricity rate, monthly cost-of-ownership explanation
- Assumptions-at-a-glance: effective throughput, active model productivity, monthly max output
- Benchmark sources (2026) for productivity multipliers
- Footer: Open source (GitHub), strt.it

## FAQ (promo / support)

**Is it free?**  
Yes. The app is free to use; the code is open source (MIT-style, see repo).

**Does it track my usage?**  
No. All inputs stay in your browser; share URLs and local profiles contain only the parameters you set.

**Why “vibe coders”?**  
Audience shorthand for developers who ship a lot with AI (Cursor, Claude, multi-window, context-heavy). The tool models that usage pattern.

**How often are prices updated?**  
Pricing and quotas are maintained in the repo (e.g. 2026 subscription/API data). Check the repo or app for “Assumptions” and benchmark sources.

**How do I add it to strt.it?**
