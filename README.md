# Vibemath

Vibemath is a frontend calculator for AI coding economics.

It compares:

- subscriptions like Cursor, Claude, Copilot, and Windsurf
- raw API providers
- self-hosted hardware
- rented GPUs from hyperscalers and marketplaces

The app lives in `web/`.

## Run

```bash
cd web
npm install
npm run dev
```

## Test

```bash
cd web
npm test
```
